ABOUT CLARO
-----------

Claro is the default administration theme for Drupal since 9.4.

It is a clean, accessible and powerful administration theme based on the 
Drupal Design System created as part of the closed Admin UI and JS modernization initiative.

ABOUT DRUPAL THEMING
--------------------

See https://www.drupal.org/docs/develop/theming-drupal for more information on Drupal 
theming.
